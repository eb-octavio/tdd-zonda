# TDD workshop
Let's learn the TDD workflow.

## Setup 🚀

1. Fork the project.

2. Clone to your local.
```
git clone git@github.com:eb-octavio/tdd-zonda.git
```
3. Create a virtualenv
```
cd tdd-zonda
mkvirtualenvwrapper --python=python3 tdd-zonda
pip install -e.[dev]
```

## Running the tests ☕️
```
pytest
```
